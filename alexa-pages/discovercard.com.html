Remember Me

Checking "Remember Me " will save your user ID on the computer you are using so you don't have to retype it every time you log in. To protect the security of your account, we can only save your user ID; we cannot save your account number.

  • To turn off this feature, just uncheck the box and your user ID will not appear the next time you log in
  • Security Tip: when you save your user ID, it's stored on the computer you are using; therefore, you should never save your user ID on a public computer (e.g., in a library or airport).
